This is the visualization suite for the Stony Brook University BOSS / INIT brain modeling tool. 

Please read the documentation found in the folder docs for more information on file formats and application use.

The source code is located in src.

The data folder is separated into model and firing. Model data defines the neural structure. Firing defines a firing series for a given set of synaptic connections over a period of time.
